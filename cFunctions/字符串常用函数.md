# 标准字符串常用函数

## 1.字符串输入输出函数

### （1）字符串标准输入：fgets()

```c
函数声明：char *fgets(char *s,int N, FILE *fp);
```

①函数参数必须是可写的字符数组或指针；

②最多读入N-1个字符，如果提前遇到了'\n'，将会结束读取（'\n'被读入数组中）结束后末尾添'\0'；

③必须指明数据来源，例如文件（FILE *fp）或键盘输入（stdin）；

④返回值：正常执行返回字符串首地址即s，出错时返回NULL。

使用示例：

```c
char test[MAXN];
fgets(test, sizeof(test), stdin);
test[strcspn(test, "\n")] = '\0';
```

### （2）字符串标准输出：puts()与fputs()

```c
函数声明：int  puts(const char *s);           //ending with    a '\n'
函数声明：int fputs(const char *s, FILE *fp); //ending without a '\n'
```

### （3）字符串格式输入：sscanf()

```
函数声明：int sscanf(const char *buf, char *format, ...);
```

函数功能：以格式化的方式从字符串中读入数据，buf为一个特定格式的字符串，保存等待分析读入的数据；format用于描述输入数据的数量和类型的字符串，随后为与格式化字符串相对应的参数表（必须为相关变量的地址&<变量>或数组）

使用示例：

```c
//Get each part from a string concerning time
char buf[] = "01/AUG/2020:12:23:24";
int day, year, ho, min, sec;
char mon[5] = "";
sscanf(buf, "%d/%3s/%d:%d:%d:%d", &day, mon, &year, &ho, &min, &sec);
```

### (4)字符串格式输出：sprintf()

```c
函数声明：int sprintf(char *buf, const char *format, ...);
```

函数功能：动态生成字符串，生成printf()所需要的格式化字符串，字符数组buf用于保存函数格式化字符串（在后续printf中使用）；format和后续变量描述输出格式的字符串，类似于printf("%d",a)中的参数。

使用示例：

```c
//calculate accurate answer of sin(x), m after dot
int m; double x; char buf[32];
scanf("%lf %d", &x, &m);
sprintf(buf, "%%.%df\n", m); 
//create a format string for printf means "%.mf\n"
printf(buf, sin(x));
```

## 2.字符串检查函数

### （1）字符串长度：strlen()

```c
函数声明：size_t strlen(const char s[]);
```

函数功能：返回字符串s中第一个'\0'前的字符数量。

函数辨析：sizeof和strlen的区别（前者计算字节数，后者计算有效字符串长度）

```c
char str1[ ] = "123"; //sizeof(str1) == 4, strlen(str1) == 3
char str2[9] = "123"; //sizeof(str2) == 9, strlen(str2) == 3
```

函数的代码实现：

```c
size_t MYstrlen(const char *s){
	const char *p = s;
    while(*p) p++;
    return p - s;
}
```

### （2）字符串中字符查找：strchr()与strrchr()

```c
函数声明：char*  strchr(const char *s, int c);
函数声明：char* strrchr(const char *s, int c);
```

函数功能：若字符串s中存在字符c，strchr()和strrchr()分别返回字符c在s中第一次和最后一次出现的位置的指针。若字符串s中不存在字符c，则返回空指针NULL。

函数的代码实现：

```c
char* MYstrchr(const char *s, int c){
	while(*s){
        if(*s == c) return (char*)s;
        s++;
    }
    if(c == '\0') return (char*)s;
    return NULL;
}
```

```c
char* MYstrrchr(const char *s, int c){
	const char *last = NULL;
    while(*s){
        if(*s == c) last = s;
        s++;
    }
    if(c == '\0') return (char*)s;
    return (char*)last;
}
```

### (3)字符串中子串查找：strstr()

```c
函数声明：char* strstr(const char *s, const char *subs);
```

函数功能：若字符串s中存在子串subs，strstr()返回子串subs在s中第一次出现的位置的指针。若字符串s中没有子串subs，则返回空指针NULL。

函数的代码实现：

```c
char* MYstrstr(const char *s, const char *subs){
	char *pos = (char*)s;
	char *s1, *s2;
	if(!*subs) return (char*)s; //*subs is empty
	while(*pos){
		//run over *s
		s1 = pos;
		s2 = (char*)subs;
		while(*s1 && *s2 && (*s1 == *s2)){
			//compare one by one
			s1++;
			s2++;
		}
		if(!*s2) return pos; //*s2 to end
		pos++;
	}
	return NULL;
}
```

## 3.字符串操作函数

### （1）字符串复制函数：strcpy()与strncpy()

```c
函数声明：char*  strcpy(char dest[], const char src[]);
函数声明：char* strncpy(char dest[], const char src[], size_t n);
```

函数功能：将src[]的字符串复制到dest[]中，并返回dest[]；strcpy()复制src[]中的全部内容，直至遇到'\0'；strncpy()复制src[]中前n个字符或者遇到'\0'；若n小于等于src的字符长度时需要手动在末尾添'\0'。

使用示例：

```c
char str[32] = {0};
strcpy(str, "this is a book");
strncpy(str, "that is not his pen", 10);
puts(str); //printf("that is not his pen")
```

函数的代码实现：

```c
char* Mystrcpy(char *dest, const char *src){
	char *p = dest;
    while ((*p++ = *src++));
    return dest;
}
```

```c
char* Mystrncpy(char *dest, const char *src, size_t n){
	size_t i;
    for(i = 0; i < n && src[i] != '\0'; ++i) dest[i] = src[i];
    for(; i < n; ++i) dest[i] = '\0';
    return dest;
}
```

### （2）字符串拼接函数：strcat()与strncat()

```c
函数声明：char*  strcat(char dest[], const char src[]);
函数声明：char *strncat(char dest[], const char src[], size_t n);
```

函数功能：将数组src[]中的字符串复制到dest[]中已有的字符串后面（dest尾部必须有'\0'，且'\0'会被抹去），并返回dest[]；strcat()复制src[]中的全部内容，直至遇到'\0'；strncat()复制src[]前n个字符或者遇到'\0'，若n小于等于src的字符长度时可能需要手动在末尾添'\0'。

函数的代码实现：

```c
char* Mystrcat(char dest[], const char src[]){
	int i = 0, j = 0;
	while(dest[i] != '\0') ++i;
	while((dest[i++] = src[j++]) != '\0');
	return dest;
}
```

```c
char* Mystrncat(char *dest, const char *src, size_t n){
	char *p = dest;
    while(*p) p++;
    while(n-- && *src) *p++ = *src++;
    *p = '\0';
    return dest;
}
```

### （3）字符串比较函数：strcmp()与strncmp()

```c
函数声明：int  strcmp(const char s1[], const char s2[]);
函数声明：int strncmp(const char s1[], const char s2[], size_t n);
```

函数功能：strcmp()判断两个字符串是否相同，strncmp()判断两个字符串前n个字符是否相同；若相同返回0，反之对当前遇到的不同的字符比较大小，若不同字符s1的ASCII编码大于s2，则返回正值，反之返回负值。

函数的代码实现：

```c
int Mystrcmp(const char s1[], const char s2[]){
	int i = 0;
	for(i = 0; s1[i] == s2[i]; ++i){
		if(s1[i] == '\0') return 0;
	}
	return s1[i] - s2[i];
}
```

```c
int Mystrncmp(const char s1[], const char s2[], size_t n){
	int i = 0;
	for(i = 0; s1[i] == s2[i] && i < n; ++i){
		if(s1[i] == '\0') return 0;
	}
	return s1[i] - s2[i];
}
```
